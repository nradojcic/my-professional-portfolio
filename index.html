My Work Samples

Project #1 - E-commerce - Shopify Custom Theme created from a Website Design

Problem: How can I create a custom Shopify store that doesn't look like every other Shopify website, and has all the options that I need? How do I bring my own HTML/CSS/JavaScript website design to Shopify platform to make my store different and unique?

Action: Developed a modern looking, landing page for a clothing store. Included video background, and animated menus, images, titles, and sale section. Added media queries for responsive design. Then converted this webpage to a Shopify theme, by transferring the code to Liquid files and Shopify JSON schemas.

Result: A Shopify theme customized to look like the original web design. Users can now customize their store themselves through Shopify Admin page. They can add or remove sections, update videos, images, titles, descriptions, theme colors. Create collections, and product pages. The theme integrates Shopify shopping cart and login options.

Project #2 - Acupuncture and Herbal Medicine Professional Clinic Website

Problem: How can I provide healthcare information to my patients? Give description of most commonly treated diseases? Establish web presence and enable potential patients to easily find the clinic location, get information about opening hours, services provided? Schedule their appointments online?

Action: Created and administered a professional TCM clinic website based on WordPress CMS. In addition to developing a responsive website, I also completed security setup - including SSL certificate, contact form protected with captcha, backup strategy, and protection against malicious attacks. And also set up the SEO, so the clinic website could be found online by search engines and potential patients.

Result: A responsive website, that provides information about Acupuncture and Herbal Medicine to the general public and the patients. Allows online contact through integrated contact form, and online self-scheduling of appointments. The website has separate sections for frequently asked questions, and pages for each of the most commonly treated diseases.
