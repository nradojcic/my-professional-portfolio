My Work

Project #1 - Professional Portfolio Webpage

Problem: How can I demonstrate some of my web development work projects online, and provide professional resume and contact options to interested clients, all in one place?

Action: Created a professional portfolio website. Used blank HTML5 Boilerplate as a starting point. Programmed the webpage using HTML5, CSS3. Added media queries to address responsive styling of the website at different screen resolutions. The website is deployed on inexpensive but effective web-hosting shared server based on Linux stack.

Result: A simple responsive landing page, containing general Resume, Technical skills, and description and links of representative projects. The webpage is fast loading, and simple to use. Adapting to various devices (desktop/laptop/phone/tablet) and different screen sizes. Providing professional looking result, with the use of fundamental principles and technologies.

Project #2 - Acupuncture and Herbal Medicine Professional Clinic Website

Problem: How can I provide healthcare information to my patients? Explain acupuncture and herbal medicine to my audience? Give description of most commonly treated diseases? Establish web presence and enable potential patients to easily find the clinic location, get information about opening hours, services provided? Schedule their appointments online?

Action: Created a professional TCM clinic website based on WordPress CMS. WordPress was selected as the implementation technology so the end user can update and add their own content - write blog posts and new pages at a later date on their own - without much need for advanced programming knowledge on their end.

In addition to developing a responsive website design, I also completed security setup - including SSL certificate, contact form protected with captcha, backup strategy, and protection against malicious attacks. And also set up the SEO, so the clinic website could be found online by search engines and potential patients.

Result: A nice and clean looking, responsive website, that provides information about Acupuncture and Herbal Medicine to the general public and the patients. Shows the clinic location on an interactive map. Allows online contact through integrated contact form, and online self-scheduling of appointments. The website has separate sections for frequently asked questions, and pages for each of the most commonly treated diseases.
